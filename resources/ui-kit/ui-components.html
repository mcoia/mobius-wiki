MOBIUS UI Kit - Component Library

This is a comprehensive component library extracted from the MOBIUS documentation mockups. All components are built with vanilla HTML and CSS, using CSS custom properties for easy theming.

Colors & Variables

The MOBIUS UI Kit uses CSS custom properties for consistent theming across all components.

Brand Colors

MOBIUS Navy

--mobius-navy
#003057

MOBIUS Teal

--mobius-teal
#0097A7

Dark Slate

--mobius-dark-slate
#2d3e50

Usage in CSS

.my-element {
    color: var(--mobius-teal);
    background: var(--mobius-navy);
    border-color: var(--gray-250);
}

Consistent heading sizes and typography system.

Heading 1 - 36px / 400 weight

Heading 2 - 28px / 400 weight (with border)

Heading 3 - 20px / 600 weight

Heading 4 - 16px / 600 weight

Layout System

Three-column responsive layout with fixed sidebars and flexible main content area.

Structure

<header class="top-header">...</header>
<div class="container">
    <nav class="left-sidebar">...</nav>
    <main class="main-content">...</main>
    <aside class="right-sidebar">...</aside>
</div>
Responsive Behavior

At 1200px: Right sidebar hides
At 768px: Columns stack vertically

Note Boxes

Styled note boxes with four variants for different message types.

Default Note This is a standard note box for general information.
Warning This is a warning note box for cautionary information.
Success This is a success note box for positive confirmations.
Danger This is a danger note box for critical warnings.

Responsive Design

The UI kit includes responsive breakpoints for optimal display on all devices.

Breakpoints

  • 1200px: Right sidebar hides
  • 768px: Columns stack vertically, reduced padding
  • 480px: Header stacks, full-width search
